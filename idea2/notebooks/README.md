# RHIE-CG Notebooks

Implemented notebooks for this idea:

1. `00_sanity_and_feature_shapes.ipynb`: generator residuals, channel counts, permutation equivariance.
2. `01_rhie_cip_visualization.ipynb`: raw/phase/RHIE/interaction heatmaps with true support overlays.
3. `02_candidate_compression_debug.ipynb`: top-K hit rate, reduction factor, score histograms.
4. `03_failure_analysis.ipynb`: checkpoint-based rerank failure gallery and residual analysis.

The scripts are the source of truth; notebooks should import from the local `idea2` modules.
